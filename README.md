---

## 🚀 Key Features

*   **Premium Visual Experience**: Clean modern layout with fluid typography pairing (**Outfit** for headings, **Inter** for body text, **JetBrains Mono** for technical specs).
*   **Interactive Design**: Dynamic card spotlight hover effects, micro-interactions, and a custom mouse-tracking radial glow.
*   **ATS & SEO Optimized**: Semantically structured DOM elements, complete metadata configurations, Alumni details, and embedded JSON-LD structured data.
*   **Highly Performant**: Pre-rendered static pages using Next.js App Router for instant load times.
*   **Custom Particle System**: Background particles powered by `particles.js` for subtle visual depth.

## ⚙️ Getting Started (Local Run)

Follow these instructions to clone the repository and start the development environment locally:

```bash
# 1. Clone the repository
git clone https://github.com/rizwanullah-dev/portfolio-website.git

# 2. Enter project folder
cd portfolio-website

# 3. Install required dependencies
npm install

# 4. Start hot-reloading development server
npm run dev
```

The application will run locally at [http://localhost:3000](http://localhost:3000).
